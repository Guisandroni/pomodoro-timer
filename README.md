# Pomodoro Timer (React + TypeScript + Vite)

Um simples e funcional timer Pomodoro construído com React, TypeScript e Styled Components.

## Tecnologias
- React 18 + TypeScript
- Vite
- Styled Components
- React Hook Form + Zod
- React Router DOM

## Scripts
- `npm run dev`: ambiente de desenvolvimento
- `npm run build`: build de produção
- `npm run preview`: preview do build
- `npm run lint`: lint do código

## Como rodar
```bash
npm install
npm run dev
```
Acesse `http://localhost:5173`.
